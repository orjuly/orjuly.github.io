git基础

发表于 2015-11-30   |   分类于 架构

本地创建分支

git checkout -b dev

本地分支推送到远程

git push origin dev

本地分支与远程分支关联

git branch --set-upstream dev origin/dev

基于远程分支创建本地分支

git checkout -b dev origin/dev

删除本地分支

git branch -d dev

删除远程分支(推送空分支到远程分支相当于删除远程分支)

git push origin :远程分支名
